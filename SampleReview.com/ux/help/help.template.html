What is SampleReview?

A very simple responsive application built on .NET, WebAPI and Angular 1.5.8

Review your favorite items (websites, products in the home list) and organize them by popularity or average rating among other columns.

While the use-case may be pretty simple, there's a lot under the hood illustrating my best-practices and development process.

You can clone this source now from GitHub, or send an email with a Microsoft email to be added to a TFS team with further details like storyboarding/backlogs, tasks, and future iterations for this example project.

Explore the source of this application illustrating:

  • A Scalable Business Project
    • A restful API built on WebAPI.
    • A T-SQL database connection & ORM built on Entity Framework.
    • An IoC implementation built on Unity; mapping "driver" projects to design-contract interfaces.
    • A simple repository pattern designed from a test-driven perspective.
    • Unit Test projects using Microsoft Testing Tools & Moq; purposfully testing individual components at every layer.
    • Clearly defined business-rule layer and traceable violations.
  • An AngularJS Application
    • A clearly organized UI, developed in a component pattern to isolate reusable UI fragments.
    • Responsive, accessible and mobile-friendly user-interface built on UI Bootstrap.
    • Clearly defined separatation of client-application and user-experience concerns.
    • Client services built on ngResource and the $resource factory, reducing code and promoting best-practice API design.
    • A NodeJS based build-pipeline built on Grunt task-runner, webpack, Sass—packing the app into an efficient module and source-mapping for easy debugging.
    • Behavior-driven JavaScript unit-tests, built on Jasmine and automated with Karma.
